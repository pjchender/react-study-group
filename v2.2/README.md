# React 前端讀書會 v2.2

> 目標：與大家一同進步，往更好的工程師前進。

本次方向：

- 閱讀 React 與前端框架相關的 Design Patterns、Rendering Patterns、Performance Patterns
  - 有可能需要自己找其他資源來補充
- 閱讀開源套件原始碼
  - 每個人挑一個、實際遇到的問題，想分享的時候提出即可
- React 18

形式：

- 依據每次分享的主題建立資料夾，並將整理且願意公開的內容放在裡面
- 整理分享 20 ~ 30 mins
- 共同討論 10mins
